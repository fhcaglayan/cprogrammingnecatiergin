#### _int_ türünün _2_ byte _long_ türünün _4_ byte olduğu bir derleyicide aşağıdaki sabitlerin türünü yazınız:

* 'A'		->  char
* 32767		->	int
* 32768		->	long int
* 50000		->	long int
* 0x7000	->	int
* 0xFFFF	->	int				->	eksi bir hepsi bir
* 50000U	->	unsigned int	
* 10000U	->	unsigned int
* '\x1A'	->	char 			->	hexadecimal representation of character
